# 🚀 DDR Report Generator

An AI-powered system that transforms raw **inspection reports** and **thermal reports** into a structured, professional **Detailed Diagnostic Report (DDR)**.

Designed for industrial use cases, this tool automates analysis, improves accuracy, and significantly reduces manual reporting effort.

---

## 📌 Key Features

- 📄 **PDF Parsing**  
  Extracts both **text and images** from inspection and thermal reports

- 🤖 **AI-Based Data Structuring**  
  Uses LLMs to intelligently extract meaningful structured insights

- 🔗 **Data Fusion Engine**  
  Merges inspection + thermal data into a unified diagnostic view

- 📊 **Automated DDR Generation**  
  Produces clean, client-ready reports instantly

- 🖥️ **Interactive UI**  
  Built with Streamlit for easy usage and visualization

---

## 🧠 Tech Stack

- **Python**
- **Streamlit**
- **PyMuPDF (fitz)** – PDF processing
- **OpenRouter API (LLM)** – AI extraction & reasoning
